

### PHP注釋

>* 注釋==代碼的解釋和說明，用來說明代碼或函數的編寫人、用途、時間等。
>* 一般放到代碼的上方或代碼的尾部（放尾部時，代碼和注釋之間以<Tab>鍵進行分隔，以方便程式閱讀）
>* 注釋不會影響到程式的執行，因為在執行時，注釋部分會被解譯器忽略不計 

PHP支持3種風格的程式注釋

[1]C++風格的單行注釋（//）
```
<?php
echo '使用C++風格';	//這就是C++風格
?>
```
[2]C風格的多行注釋（/*…*/）
```
<?php
/*C
風格的
多行注釋
*/
echo '只會看到這句話。';
?>
```
注意：多行注釋是不允許進行嵌套操作的。

[3]Shell風格的注釋（#）
```
<?php
echo '這是Shell腳本風格的注釋';		#這裡的內容是看不到的
?>
```
注意：在單行注釋中的內容不要出現“?>”標誌，因為解譯器會認為PHP腳本結束，而去執行“?>”後面的代碼
